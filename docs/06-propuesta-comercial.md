# Propuesta Comercial - ML Scouting Academy

## Contexto

ML Scouting Academy tiene su presencia digital dispersa en 6 plataformas distintas (ml-scouting.com, TiendUp, Gumroad, Tally, Google Forms, Calendly). Esto genera:
- Friccion en la experiencia del usuario
- Perdida de leads potenciales
- Imagen de marca inconsistente
- Dificultad para hacer seguimiento de clientes

---

## Que Ofrecemos

### Landing Page Profesional Todo-en-Uno

Una unica plataforma web que consolida toda la operacion digital de ML Scouting:

**1. Presencia de marca profesional**
- Diseño personalizado con identidad de ML Scouting
- Logo, colores y tipografia consistentes
- Video hero de alto impacto
- Seccion del fundador con credenciales

**2. Servicios integrados**
- Los 4 servicios para jugadores presentados profesionalmente
- Tabla comparativa para que el cliente elija facil
- CTAs directos a pago (Gumroad) o agendar reunion (Calendly)

**3. Diagnostico inteligente (reemplaza Tally)**
- Formulario interactivo integrado en la web
- Recomendacion automatica del servicio ideal
- El jugador no necesita ir a Instagram para continuar
- Captura de datos del lead

**4. Curso de Scouting con lista de espera**
- Informacion completa de Fase 1 y Fase 2
- Formulario de lista de espera integrado (reemplaza Google Forms)
- Testimonios y resenas reales

**5. Blog y SEO**
- Articulos para posicionamiento en buscadores
- Contenido de valor que atrae trafico organico

**6. Contacto profesional**
- Formulario integrado
- WhatsApp directo
- Redes sociales

## Opciones de Desarrollo

### Opcion A: Landing Page Estatica
- React + Tailwind (lo que ya esta prototipado)
- Deploy en hosting propio o Vercel/Netlify
- Links externos a Gumroad para pagos
- Formularios con envio a email
- **Ideal para:** arrancar rapido con bajo costo

### Opcion B: Landing + Backend Basico
- Todo lo de Opcion A
- Base de datos para leads del diagnostico y lista de espera
- Panel admin simple para ver leads
- Notificaciones por email cuando llega un lead
- **Ideal para:** tener control de los datos sin depender de Google Forms

### Opcion C: Plataforma Completa
- Todo lo de Opcion B
- Pasarela de pago propia (Stripe/MercadoPago)
- Dashboard de leads y clientes
- Email marketing automatizado
- Area privada para alumnos del curso
- **Ideal para:** independizarse totalmente de Gumroad/TiendUp
