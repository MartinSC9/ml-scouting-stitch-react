# Propuesta: Nueva Landing Page para ML Scouting Academy

## Resumen Ejecutivo

ML Scouting Academy necesita consolidar su presencia digital en una sola plataforma profesional. Actualmente, la informacion y servicios estan dispersos en multiples herramientas que no estan integradas entre si, lo que dificulta la experiencia del usuario y la captacion de clientes.

**Objetivo:** Crear una landing page profesional que unifique todos los servicios, el curso de scouting y la captacion de leads en un solo lugar, con una experiencia fluida para el usuario.

---

## Problema Actual

| Herramienta | Uso actual | Problema |
|---|---|---|
| ml-scouting.com | Landing principal | No integra servicios ni pagos |
| TiendUp | Venta de cursos | Tienda generica, no transmite la marca |
| Gumroad | Venta de servicios (4 productos) | Plataforma externa, el cliente sale de la web |
| Tally | Diagnostico para jugadores | Formulario externo, no integrado |
| Google Forms | Lista de espera del curso | Proceso manual, datos en un Doc |
| Calendly | Agendar reuniones | Link externo |
| Instagram | Contacto post-diagnostico | Informal, sin seguimiento |

**Resultado:** El cliente potencial tiene que navegar entre 5+ plataformas para entender que ofrece ML Scouting, elegir un servicio y pagar. Esto genera friccion y perdida de oportunidades.

---

## Solucion Propuesta

Una landing page unica y profesional que incluya:

### 1. Home con identidad de marca
- Hero con video de fondo y propuesta de valor clara
- Seccion "Sobre el Fundador" con credenciales de Marco Lujan
- Servicios resumidos con CTAs directos
- Testimonios reales de alumnos
- Seccion del curso con lista de espera integrada

### 2. Pagina de Servicios
- Los 4 servicios actuales de Gumroad presentados profesionalmente
- Tabla comparativa de servicios
- Seccion destacada del Plan de Mercado (mas popular)
- Flujo: ver servicio → contratar (link a Gumroad o pago integrado)

### 3. Diagnostico Interactivo (reemplaza Tally)
- Las mismas preguntas del formulario actual pero con UX profesional
- Recomendacion automatica del servicio ideal segun respuestas
- CTA directo a la contratacion del servicio recomendado
- Captura de datos del lead sin depender de Instagram

### 4. Curso de Scouting
- Informacion completa del curso (Fase 1 y Fase 2)
- Formulario de lista de espera integrado (reemplaza Google Forms)
- Testimonios especificos del curso
- Proxima edicion: Septiembre 2026

### 5. Blog
- Articulos sobre scouting, mercado y desarrollo profesional
- Posicionamiento SEO

### 6. Contacto
- Formulario de contacto directo
- Links a redes sociales
- WhatsApp directo

---

## Flujo del Usuario

```
Landing Page
    |
    ├── Soy Jugador → Diagnostico → Recomendacion de servicio → Contratar
    |
    ├── Soy Club → Contacto directo / Asesoria para clubes
    |
    └── Quiero ser Scout → Info del curso → Lista de espera
```

## Ventajas vs. Situacion Actual

| Aspecto | Antes | Despues |
|---|---|---|
| Presencia online | 5+ plataformas dispersas | 1 landing profesional |
| Identidad de marca | Generica (TiendUp) | Profesional y personalizada |
| Captacion de leads | Manual (Forms + Instagram) | Automatizada con diagnostico |
| Experiencia del usuario | Confusa, multiples saltos | Fluida, todo en un lugar |
| Servicios | Escondidos en Gumroad | Presentados con claridad |
| Lista de espera | Google Forms suelto | Integrada en la web |
| SEO | Minimo | Blog + landing optimizada |
